# task-manager
A simple and beginner-friendly Task Manager App built in Python, which allows users to add, view, and remove tasks using lists and functions.  Designed to demonstrate the use of functions, loops, and lists in Python, this project is ideal for students and beginners learning about basic data handling and menu-driven applications.
